# Happy Marriage Skills

[English](README.en.md) | 简体中文

基于约翰·戈特曼博士40年研究的《幸福的婚姻》一书，帮助伴侣建立持久、充实的亲密关系。

## 适用场景

* **关系健康检查** - 评估关系状态，识别改进领域
* **冲突解决** - 处理分歧，学习建设性对话
* **增强连接** - 建立更深的情感纽带
* **亲密关系学习** - 学习亲密关系知识，为将来做准备
* **危机干预** - 处理严重关系困扰

## 安装技能

```bash
npx skills add https://github.com/happy-marriage/skills
```
