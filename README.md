# Happy Marriage Skills

[English](README.en.md) | 简体中文

基于约翰·戈特曼博士40年研究的《幸福的婚姻》一书，帮助伴侣建立持久、充实的亲密关系。

## 适用场景

- **关系健康检查** - 评估关系状态，识别改进领域
- **冲突解决** - 处理分歧，学习建设性对话
- **增强连接** - 建立更深的情感纽带
- **危机干预** - 处理严重关系困扰

## 安装技能

```bash
npx skills add https://github.com/happy-marriage/skills
```

## 使用示例

> 以下示例来源于小红书

- 我老婆垃圾不放在垃圾桶里，用完的东西不会放回原位，我天天跟着屁股后面收拾，也没有被得到承认，反而因为我提醒她几句，她就要搬出去住。我要怎么办？
- 和女朋友快一年了，什么节日我都准备礼物，日常开支都是我出，要买啥买啥。可是从来我就没收过她买的一份礼物（淘宝买东西的问我要什么凑单，然后我代付），生气了就不说话。今天七夕我送了礼物，因为餐厅人多没订到位，随便去茶餐厅对付了一口，就生气了，这怎么办？
- 十几年前我哭着给前夫打电话说：你妈和你姐欺负我...，我前夫来了句：那是我妈那是我姐！她们就那样！事后我妈好声好气的教他说：做老公的在妈妈和媳妇中间要学会做和事佬，他来句：媳妇可以换，妈换不了！所以结婚一年半我们就离婚了……
