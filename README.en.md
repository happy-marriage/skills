# Happy Marriage Skills

Based on Dr. John Gottman's 40 years of research from "The Seven Principles for Making Marriage Work," this skill helps couples build lasting, fulfilling intimate relationships.

## Use Cases

* **Relationship Health Check** - Assess relationship status and identify areas for improvement
* **Conflict Resolution** - Handle disagreements and learn constructive dialogue
* **Strengthen Connection** - Build deeper emotional bonds
* **Intimate Relationship Learning** - Learn about intimate relationships to prepare for the future
* **Crisis Intervention** - Address serious relationship distress

## Install the Skill

```bash
npx skills add https://github.com/happy-marriage/skills
```
